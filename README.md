# Blend-filament
I am experimenting with creating blended filaments by combining two or more colors to expand the range of color expression in 3D printing. Since the resulting colors can differ from both web colors and process colors, I plan to document the actual outcomes.

# 混色フィラメント研究ノート

白×黒の混色フィラメントについて、
ノズル径・流量・モデル形状ごとの色ムラを定量化するプロジェクト。

## 現在の進捗
- ImageJ を使った明度標準偏差の測定
- 3DBenchy のノズル比較
- スウォッチ設計（24×24×2mm）
