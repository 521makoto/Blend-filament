混色フィラメント研究ノート（Multicolor Blend Filament Research）

本プロジェクトは、FDM 3Dプリンタで使用される「フィラメントカラー」の特性を体系的に記録・分析するための研究ノートです。

既に研究が進んでいる Web カラー（RGB）やプロセスカラー（CMYK）とは異なり、  
**フィラメントカラーは材料特性・透過率・粒子分布・押出条件など多くの要因が絡み合うため、色の再現性や混色挙動が未解明の部分が多い**という特徴があります。

本リポジトリでは、これらの未知の領域を実験と測定によって明らかにし、  
「3Dプリントにおける色の理解と再現性向上」を目指します。
・・・というところを目指したいけれど、今は個人メモ

## 🎯 目的（Purpose）

- Web カラーやプロセスカラーとは異なる **フィラメント固有の色挙動**を明らかにする
- 色材の混合比・ノズル径・流量・層高・モデル形状などが色に与える影響を体系化
- 縦長で多数の混色比を一括で印刷できるテストモデル「**Blend Color Checker**」を設計
- 混色フィラメントの再現性・予測性を高めるための基礎データの蓄積
- 既存の色空間（RGB/CMYK/Lab）とフィラメントカラーの関係性を探る

## 📦 研究内容（Contents）

1. ノズル径・流量ごとの比較

- 0.2 / 0.4 / 0.6 / 0.8 mm（標準）
- 0.4 / 0.6 / 0.8 mm（高流量）
- 3DBenchy やスウォッチを用いた色ムラの比較
- ImageJ による明度・色差の標準偏差測定

2. 混合比と色の関係

- **Blend Color Checker**による任意の色同士の混合比ごとの色変化
- カラーセンサーによる L*a*b* 値の記録
- 画像解析による均一度評価

3. 測定方法

- 撮影条件の統一（照明・距離・角度・露出）
- ImageJ（Fiji）による ROI 設定と標準偏差の算出
- カラーセンサーの測定手順
- データ整理のルール

4. Blend Color Checker設計

- 17×17×178mm の測定用タワー
- 縦に押し出されるフィラメントは印刷方向によって色の混ざり方が分かる。
　→ タワーの4面を測定できるような構造とする。
- 大幅な時間短縮
　　・従来：（フィラメント印刷1時間＋モデル印刷1時間）×35パターン＝70時間
  　・Checker使用：（フィラメント印刷1.3時間＋モデル印刷1.2時間）×2モデル(Standard, Detail)=5時間(93%時間短縮)

## 📊 評価指標（Metrics）

● 明度標準偏差（ImageJ）

- ROI 内の明度のばらつきを数値化
- 小さいほど均一、大きいほどまだらが強い

● 色差（ΔE）

- L*a*b* の差分を用いた色の変化量
- 混合比の変化を追跡するための指標

● カラーセンサー L*a*b* 値

- 色の平均値を比較
- 色の安定性・再現性を評価

● 目視評価（補助）

- 大面積（船体側面）
- 小面積（キャビン側面）
- 曲面部（船首・船尾）
- 細部（煙突まわり）

## 🧪 使用ツール（Tools）

- **Fiji / ImageJ**：明度・色差の標準偏差測定
- **カラーセンサー**：L*a*b* 値の取得
- **Bambu Lab X1C / P1 系列**：造形
- **3DBenchy / スウォッチ**：比較用モデル
- **撮影環境**：照明・背景・固定治具
